# Git Basics Cheatsheet

```bash
cd meinprojekt
ls -la

git init .
git status

git add index.html
git status

git commit -m "Initial commit"
git status --short

git add css/
git add .  # "." entspricht dem aktuell offenen Ordner
git commit -m "Add assets"
git status

```
